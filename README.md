# 쁠리 후원 랭킹 (핑크 테마, 링크+카테고리+세부접기)

- **닉네임 클릭 → https://www.sooplive.co.kr/station/<아이디>** 새 탭 열림
- 탭: **총후원개수순 / 참여율순 / 금액순**
- 행 확장: ▸ 버튼으로 **회차/내역/요약/메모/설명** 등 세부 필드 표시
- 기본 테이블은 세부 필드 감춤, 모바일 최적화

## 배포
1. `index.html`, `style.css`, `script.js`, `README.md`를 저장소에 업로드(덮어쓰기).
2. Settings → Pages에서 main/(root) 설정.
3. `https://<유저명>.github.io/<저장소명>/` 접속.
