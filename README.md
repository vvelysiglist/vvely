# 쁠리 후원 랭킹 (순위/메뉴/회차정렬 반영판)

## 메뉴
1. **누적후원랭킹**: `순위 / 닉네임 / SOOP / 총후원개수 / 참여횟수 / 참여율`
   - 총후원 옆 **□** 버튼 → 회차별 후원개수 접기/펼치기
2. **참여율랭킹**: `순위 / 닉네임 / SOOP / 참여율`
3. **회차별목록**: 상단 6개 회차 버튼 → 해당 회차에서 `순위 / 닉네임 / SOOP / 후원개수`

- 닉네임은 링크 X, 옆 **SOOP** 버튼을 클릭하여 `https://www.sooplive.co.kr/station/<id>` 이동
- 하위 6개 회차는 라벨의 날짜/회차를 파싱하여 **오래→최근** 정렬

## 배포
1. `index.html`, `style.css`, `script.js`, `README.md`를 저장소에 업로드(덮어쓰기).
2. Settings → Pages에서 main/(root) 설정.
3. `https://<유저명>.github.io/<저장소명>/` 접속.
