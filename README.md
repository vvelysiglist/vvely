# 쁠리 후원 랭킹 뷰어 (PRO v2)

새로 업로드된 엑셀을 반영한 버전입니다. 모바일 최적화 + 테마 + 컬럼 숨김/표시 기능 포함.

## 배포 방법
1. GitHub 저장소에서 **Add file → Upload files**로 아래 파일 4개를 업로드(덮어쓰기)합니다.
   - `index.html`, `style.css`, `script.js`, `README.md`
2. Settings → Pages에서 **Branch: main / root** 설정 확인.
3. `https://<유저명>.github.io/<저장소명>/`에서 확인.

## 기능
- 헤더 클릭 정렬 (숫자 큰순 기본 추정)
- 검색, 시트 필터
- **테마 변경 (보라/민트/핑크/골드/블루)**
- **컬럼 숨김/표시 토글**

## 데이터 갱신
엑셀 원본이 바뀌면 이 대화로 새 파일 생성 요청 주세요. 즉시 갱신 파일 만들어드립니다.
