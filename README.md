# 쁠리 후원 랭킹 뷰어

GitHub Pages에 바로 올릴 수 있는 정적 사이트입니다.

## 배포 방법
1. GitHub에서 새 저장소를 만들고, `main` 브랜치에 이 폴더의 파일들을 업로드합니다.
2. Settings → Pages에서 **Branch: main / root** 선택 후 저장합니다.
3. 몇 분 후 `https://<사용자명>.github.io/<저장소명>/` 주소에서 확인 가능합니다.

## 데이터 갱신
- 새로운 엑셀로 교체하려면, 로컬에서 `index.html`과 `script.js`에 내장된 데이터 부분을 다시 생성해야 합니다(제가 도와드릴 수 있어요).
