# 쁠리 후원 랭킹 (메뉴형)

- 메인 메뉴 3가지
  1. **누적후원랭킹**: 총후원개수 내림차순
  2. **참여율랭킹**: 참여율 내림차순
  3. **회차별목록**: 감지된 상위 6개 회차 버튼 → 해당 회차의 후원개수 순 정렬
- 닉네임 클릭 → `https://www.sooplive.co.kr/station/<id>` 새 탭
- 밝은 핑크 그라데이션 테마

## 배포
1. `index.html`, `style.css`, `script.js`, `README.md`를 저장소에 업로드(덮어쓰기).
2. Settings → Pages에서 main/(root) 설정.
3. `https://<유저명>.github.io/<저장소명>/` 접속.
