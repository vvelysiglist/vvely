# 쁠리 후원 랭킹 (닉 옆 SOOP / 참여율 보강판)

## 메뉴 구성
1. **누적후원랭킹**: `순위 / 닉네임(+SOOP) / 총후원개수(□) / 참여횟수 / 참여율`
2. **참여율랭킹**: `순위 / 닉네임(+SOOP) / 참여율 / 참여횟수 / 총후원개수(□)`
3. **회차별목록**: 선택한 회차의 `순위 / 닉네임(+SOOP) / 후원개수`

- **□**: 클릭 시 해당 닉네임의 최근 6개 회차별 후원개수 표시
- 닉네임은 링크가 아니며, 옆의 **SOOP** 버튼으로 `https://www.sooplive.co.kr/station/<id>` 이동
- 회차 버튼은 라벨의 날짜/회차 파싱 기준으로 **오래→최근** 6개 노출

## 배포
1. `index.html`, `style.css`, `script.js`, `README.md`를 저장소에 업로드(덮어쓰기).
2. Settings → Pages에서 main/(root) 설정.
3. `https://<유저명>.github.io/<저장소명>/` 접속.
