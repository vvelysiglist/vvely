# 쁠리 후원 랭킹 (핑크 테마, 카테고리+세부접기)

- 탭: **총후원개수 / 참여율 / 금액** 기준 랭킹 전환
- 행 확장: ▸ 버튼 클릭 시 **회차/내역/요약/메모/설명** 등 세부 필드 표시
- 기본 테이블에서는 세부 필드는 감춰 깔끔하게 표시
- 검색, 시트 필터, 컬럼 토글, 모바일 최적화

## 배포
1. `index.html`, `style.css`, `script.js`, `README.md`를 저장소에 업로드(덮어쓰기).
2. Settings → Pages에서 main/(root) 설정.
3. `https://<유저명>.github.io/<저장소명>/` 접속.
