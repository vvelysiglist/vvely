# 쁠리 후원 랭킹 (SOOP SVG 로고 적용)

- 닉네임 옆 **SOOP 로고**로 `https://www.sooplive.co.kr/station/<id>` 새 탭 이동
- 메뉴
  1. **누적후원랭킹**: `순위 / 닉네임(+로고) / 총후원개수(□) / 참여횟수 / 참여율`
  2. **참여율랭킹**: `순위 / 닉네임(+로고) / 참여율 / 참여횟수 / 총후원개수(□)`
  3. **회차별목록**: 선택 회차의 `순위 / 닉네임(+로고) / 후원개수`

배포: 4개 파일(`index.html`, `style.css`, `script.js`, `README.md`)을 저장소에 업로드하고 Pages를 활성화하세요.
